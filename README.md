**AI-powered decision making assistant**

Mobile app that helps you make difficult choices through AI-generated questions and personalized recommendations.

[![Get it on Google Play](https://play.google.com/intl/en_us/badges/static/images/badges/en_badge_web_generic.png)](https://play.google.com/store/apps/details?id=com.pickai)

## ‚ú® How It Works

1. **Enter Your Options** - Input the choices you're deciding between
2. **Answer AI Questions** - Respond to 7-10 personalized questions generated by AI  
3. **Get Recommendation** - Receive detailed explanation of which option suits you best

## üöÄ Features & Functionality

### Core Features
- **AI Question Generation** - Dynamic question creation based on user's decision options
- **Multi-Model AI Support** - Integration with multiple AI models (GPT, Deepseek, Gemma, Mistral, Nemotron)

### Technical Implementation
- **State Management** - Zustand for lightweight, efficient state handling
- **Navigation System** - React Navigation 7 with fade animations and headerless design
- **Remote Configuration** - Firebase Remote Config for dynamic app settings
- **Environment Management** - Secure API key handling with react-native-config
- **TypeScript Integration** - Full type safety across the entire codebase

### Monetization & Analytics
- **Ad Integration** - Appodeal SDK with banner, interstitial, and rewarded video ads
- **Remote Model Configuration** - Dynamic AI model switching without app updates

### User Experience
- **Intuitive UI** - Clean, minimalist interface with custom components
- **Loading States** - Proper pending state management during AI processing
- **Error Handling** - Graceful error management and user feedback
